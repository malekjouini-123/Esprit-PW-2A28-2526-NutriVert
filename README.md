# NutriVert CRUD Final

Projet PHP MVC + OOP + PDO pour les entités `recette` et `instruction`.

## Fonctionnalités
- FrontOffice : liste des recettes + détail d'une recette
- BackOffice : CRUD recette + CRUD instruction
- Création complète recette + plusieurs étapes d'instruction dans la même page
- Contrôle de saisie JS + PHP
- Affichage des erreurs sous les champs
- Affichage des ingrédients décodés depuis JSON : nom, quantité, image

## Lancement sous XAMPP
1. Copier le dossier dans `C:\xampp\htdocs\nutrivert_crud_final`
2. Démarrer Apache et MySQL
3. Créer la base `nutrivert`
4. Importer `database/nutrivert.sql`
5. Ouvrir `http://localhost/nutrivert_crud_final/public/`
